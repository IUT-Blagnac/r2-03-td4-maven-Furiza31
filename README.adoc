= Compiler et construire ses applications avec maven
Hugo Wendjaneh

== Nettoyer tout ce qui a pu se passer avant
....
mvn clean
....

== Lancer la compilation
....
mvn compile
....

== Lancer la compilation en jar
....
mvn package
....

== Lancer le .jar
....
java -jar target/tp_qualite-1.0.jar
....

== Générer la javadoc
....
mvn javadoc:javadoc
....
